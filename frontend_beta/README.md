# Frontend (frontend_beta)

La documentacion principal para ejecutar el sistema completo (API + frontend) esta en el README raiz del repositorio:

- `../README.md`

## Comandos rapidos (solo frontend)

```powershell
cd .\frontend_beta
npm install
npm run dev
```

URL local:

- `http://localhost:5173`

Nota:

- Este frontend depende de la API corriendo en `http://localhost:5062` via proxy de Vite (`/api`).
